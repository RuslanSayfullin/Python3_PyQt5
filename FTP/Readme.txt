                        Описание электронного архива
            к книге "Python 3 и PyQt 5: разработка приложений, 2 изд."
----------------------------------------------------------------------------
----------------------------------------------------------------------------

В состав архива включены следующие файлы и каталоги:

 - sudoku - каталог с исходными текстами приложения "Судоку". Для запуска
   приложения следует запустить файл start.pyw, находящийся непосредственно
   в этом каталоге;

 - Listings.doc - все листинги из книги;

 - PyQt.doc - более 750 дополнительных листингов, поясняющих материал из
   второй части книги, посвящённой PyQt;

 - Readme.txt - этот файл.
